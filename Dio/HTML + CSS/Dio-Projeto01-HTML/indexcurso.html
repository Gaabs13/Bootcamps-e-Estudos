O que é HTML5?

A internet é construída sobre muitas camadas de tecnologia, e uma das principais é o HTML.

Vamos explorar o que é o HTML5, como estruturar um documento HTML de forma eficiente e a importância do HTML semântico. No final, falaremos sobre uma ferramenta incrível chamada NotebookLM, que pode transformar sua forma de estudar programação.

Como Estruturar um Documento HTML

Anatomia Básica do HTML

  • Nó Document: representa o documento inteiro.
  • Elementos (tags): como <body>, <p>, <img>.
  • Texto: conteúdo textual dos elementos.
  • Comentários: notas internas no código.

Partes Essenciais de um Documento HTML

  • <head>: contém informações sobre o documento (metadados, título, links para CSS).
  • <body>: onde fica o conteúdo visível da página.

Tipos de Elementos HTML

  • Inline (em linha): exemplos: <span>, <a>, <strong>.
  • Block-level (em bloco): exemplos: <div>, <section>, <article>.

Como Estilizar e Manipular um HTML com CSS e JavaScript

Estilização com CSS

  • Estilo em linha: usado diretamente no elemento (ex: <p style="color:red;">).
  • Folhas de estilo internas: dentro da tag <style> no <head>.
  • Folhas de estilo externas: em arquivos separados com extensão .css, vinculados via <link>.

Manipulação com JavaScript

  • Selecionar elementos: usando métodos como getElementById(), getElementsByClassName() ou querySelectorAll().
  • Modificar conteúdo e atributos: exemplo: alterar o texto de um parágrafo com innerHTML ou mudar a imagem exibida alterando o atributo src.
  • Criar e inserir elementos dinamicamente: com métodos como createElement() e appendChild().
  • Como incluir JavaScript:
    • Diretamente no HTML usando <script>.
    • Através de arquivos externos (exemplo: <script src="script.js"></script>).
    • Como manipuladores de eventos inline (menos recomendado atualmente).

O que é HTML Semântico?

HTML Semântico significa usar tags que descrevem o significado do conteúdo.

Exemplos de Elementos Semânticos

  • Estruturais: <header>, <footer>, <main>, <section>, <article>, <aside>, <nav>.
  • Conteúdo específico: <figure>, <figcaption>, <mark>, <time>, <details>, <summary>.

Por que Usar HTML Semântico?

  • Melhora a acessibilidade: facilita a vida de quem usa leitores de tela.
  • Aumenta o SEO: mecanismos de busca entendem melhor o conteúdo.
  • Facilita a manutenção: o código fica mais legível e organizado.
  • Separa estrutura de estilo: o conteúdo fica independente da forma como será exibido.

Dica: Embora elementos como <div> e <span> sejam muito usados, eles não são semânticos.
