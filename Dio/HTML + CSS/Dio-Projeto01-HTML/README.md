# 📄 Introdução ao HTML5 e HTML Semântico

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte dos estudos de **HTML5**, com o objetivo de explorar os principais conceitos da linguagem de marcação utilizada na construção de páginas web.

A aplicação apresenta uma página informativa que aborda desde a estrutura básica de documentos HTML até conceitos mais avançados, como **HTML Semântico**, integração com **CSS** e **JavaScript**, acessibilidade, organização de conteúdo e boas práticas de desenvolvimento web.

---

## 🎯 Objetivos de Aprendizagem

- Compreender a estrutura fundamental de um documento HTML.
- Utilizar corretamente elementos semânticos do HTML5.
- Organizar conteúdos utilizando listas, links e seções.
- Aplicar conceitos básicos de estilização com CSS.
- Entender a integração entre HTML, CSS e JavaScript.
- Desenvolver páginas mais acessíveis e otimizadas para SEO.

---

## 🛠️ Tecnologias Utilizadas

- HTML5

## 📚 Conceitos Trabalhados

### Estrutura HTML

- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<body>`
- Metadados
- Hierarquia de títulos

### Elementos de Conteúdo

- Parágrafos
- Listas ordenadas e não ordenadas
- Links internos e externos
- Citações (`blockquote`)
- Destaques (`mark`)
- Abreviações (`abbr`)

### HTML Semântico

- `<header>`
- `<main>`
- `<section>`
- `<article>`
- `<nav>`
- `<footer>`
- `<aside>`
- `<figure>`
- `<figcaption>`

### Estilização

- CSS interno
- Tipografia personalizada
- Organização visual de conteúdo
- Responsividade básica
