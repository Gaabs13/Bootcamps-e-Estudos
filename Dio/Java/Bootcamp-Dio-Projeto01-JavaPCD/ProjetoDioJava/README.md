# 📌 Projeto DIO + Bradesco PCD Tech Java & QA Developer - Projeto Introdutório

Projeto desenvolvido em Java, com foco na aplicação de métodos de teste e depuração para identificação, análise e correção de bugs no código. O objetivo principal é compreender o comportamento da aplicação, interpretar erros e aplicar correções de forma estruturada e eficiente.

**O projeto aborda conceitos fundamentais como:**

Leitura e interpretação de erros de execução

Validação de comportamentos esperados

Organização do código para facilitar testes e manutenção

Aplicação do debug como parte do processo de desenvolvimento

**🧪 Testes e Debug**

Durante o desenvolvimento, foram utilizados métodos de teste manuais e análise de fluxo de execução para:

Identificar falhas lógicas

Corrigir inconsistências no código

Melhorar a previsibilidade dos resultados

Reforçar boas práticas de desenvolvimento em Java

Esse processo contribuiu para o fortalecimento do pensamento analítico, da lógica de programação e da compreensão da importância dos testes no ciclo de desenvolvimento de software.

## 🎓 Contexto do Projeto

Projeto desenvolvido no contexto do programa DIO PCD, com foco em aprendizado prático e acessível.
Serviu como base conceitual e técnica para projetos posteriores, como o Beerstock, ampliando a maturidade no desenvolvimento e na resolução de problemas.

**🚀 Tecnologias Utilizadas**

Java

Programação Orientada a Objetos

Testes e Debug de código

**📈Importância dos testes no processo de desenvolvimento:**

Análise de erros e correção de bugs

Organização e legibilidade do código

Evolução do raciocínio lógico aplicado à programação
